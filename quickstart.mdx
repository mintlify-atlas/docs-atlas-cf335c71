Get up and running with Trezor Suite quickly, whether you're a user setting up your hardware wallet or a developer contributing to the project.

<Tabs>
  <Tab title="For Users">
    ## Using Trezor Suite

    Get started with Trezor Suite to manage your cryptocurrency portfolio securely.

    <Steps>
      <Step title="Get a Trezor Device">
        Purchase a Trezor hardware wallet from [trezor.io](https://trezor.io).
        
        **Supported models:**
        - Trezor Model T
        - Trezor Model One
        - Trezor Safe 3
        - Trezor Safe 5
      </Step>

      <Step title="Download Trezor Suite">
        Choose your platform:
        
        - **Desktop:** Download from [suite.trezor.io](https://suite.trezor.io)
        - **Web:** Visit [suite.trezor.io/web](https://suite.trezor.io/web) in your browser
        - **Mobile:** Download from App Store or Google Play
        
        See [Installation](/installation) for detailed instructions.
      </Step>

      <Step title="Set Up Your Device">
        1. Connect your Trezor device to your computer or mobile device
        2. Follow the onboarding wizard in Trezor Suite
        3. Create a new wallet or recover an existing one
        4. Write down your recovery seed and store it safely
        
        <Warning>Never share your recovery seed with anyone!</Warning>
      </Step>

      <Step title="Add Your Accounts">
        1. Enable cryptocurrencies you want to use
        2. Suite will discover your accounts automatically
        3. Start receiving and sending cryptocurrency
        
        Learn more about [Account Discovery](/suite/account-discovery).
      </Step>
    </Steps>

  <Tab title="For Developers">
    ## Development Setup

    Set up your development environment to contribute to Trezor Suite.

    <Steps>
      <Step title="Prerequisites">
        Install required tools:
        
        ```bash
        # Install NVM (Node Version Manager)
        curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
        
        # Install Node.js 24
        nvm install 24
        nvm use 24
        
        # Enable Yarn
        corepack enable
        
        # Install Git LFS
        # macOS: brew install git-lfs
        # Ubuntu: sudo apt-get install git-lfs
        ```
        
        See [Environment Setup](/development/environment-setup) for platform-specific instructions.
      </Step>

      <Step title="Clone Repository">
        ```bash
        git clone git@github.com:trezor/trezor-suite.git
        cd trezor-suite
        git submodule update --init --recursive
        git lfs install
        git lfs pull
        ```
      </Step>

      <Step title="Install Dependencies">
        ```bash
        # Install Node.js version specified in .nvmrc
        nvm install
        
        # Install dependencies (takes 5-10 minutes)
        yarn
        
        # Build essential packages (takes 5-10 minutes)
        yarn build:essential
        ```
      </Step>

      <Step title="Run Development Server">
        Choose what to run:
        
        <CodeGroup>
        ```bash Suite Web
        yarn suite:dev
        ```
        ```bash Suite Desktop
        yarn suite:dev:desktop
        ```
        ```bash Suite Mobile
        yarn native:start
        ```
        </CodeGroup>
        
        See [Running Apps](/development/running-apps) for more options.
      </Step>
    </Steps>

    ## Integrate Trezor Connect

    Add Trezor hardware wallet support to your application:

    <CodeGroup>
    ```bash npm
    npm install @trezor/connect-web
    ```
    ```bash yarn
    yarn add @trezor/connect-web
    ```
    ```bash pnpm
    pnpm add @trezor/connect-web
    ```
    </CodeGroup>

    ```typescript
    import TrezorConnect from '@trezor/connect-web';

    // Initialize
    await TrezorConnect.init({
      manifest: {
        email: 'developer@example.com',
        appUrl: 'https://example.com',
      },
    });

    // Get Bitcoin address
    const result = await TrezorConnect.getAddress({
      path: "m/49'/0'/0'/0/0",
      coin: 'btc',
    });

    if (result.success) {
      console.log('Address:', result.payload.address);
    }
    ```

    Learn more in the [Connect SDK docs](/connect/overview).
